# Tiles Visualizer

## Set Up
1. Clone the repository
2. Create a virtual environment (OPTIONAL) :

    ```
    python3 -m venv env
    ```
3. Activate the environment (OPTIONAL):
    ```
    source env/bin/activate
    ```
4. install pip dependencies:
    ```
    pip install -r requirements.txt
    ```
## Running

Just type:

```
python main.py
```
